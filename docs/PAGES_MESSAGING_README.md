## üí° Pro Tips

### Maximize Approval Chances

**1. Over-communicate**
- Longer description is better than shorter
- More test instructions better than fewer
- Show more in video than you think necessary

**2. Make it easy for reviewers**
- Test credentials should work first try
- Test Page should have obvious messages
- Video should be self-explanatory
- Each step should be crystal clear

**3. Follow the script**
- Don't improvise the video
- Don't skip verification step
- Don't rush through demo
- Don't assume reviewer knows anything

**4. Test everything yourself**
- Log out and test with provided credentials
- Watch your video with fresh eyes
- Read description as if you're the reviewer
- Follow test instructions exactly as written

## ‚ö†Ô∏è Critical Mistakes to Avoid

### ‚ùå Don't Do This:

1. **Using test user from App Roles**
   - Test users can't receive messages
   - Use real Facebook account instead

2. **Skipping Facebook verification in video**
   - Must show reply appears in Messenger
   - This proves integration works

3. **Rushing the video**
   - Slow down and show each step
   - Pause to let reviewer read text

4. **Incomplete description**
   - Don't just say "for messaging"
   - Explain how, why, value, compliance

5. **Wrong test credentials**
   - Double-check email/password
   - Test them yourself first

6. **No test data**
   - Must have real messages in Inbox
   - Send messages before recording

7. **Poor video quality**
   - Must be 1080p minimum
   - Text must be readable
